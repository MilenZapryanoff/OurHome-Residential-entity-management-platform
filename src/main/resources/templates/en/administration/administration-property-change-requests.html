Individual units management


This section provides information about all individual properties that are part of the Digital Condominium.

Here, you can manage all Digital unit data change requests that require verification by the Condominium manager.

In the 'Property categories' menu, you can create a categorization of the individual properties within the Digital Condominium, which will allow you to apply different rates when calculating the monthly fees.

Property data change requests


The register contains information about all pending requests for changes to the parameters of digital properties.

These requests usually concern changes in the number of occupants in an individual property. Keeping this information up to date can be essential when calculating the monthly fees.

To apply the changes to a specific property, you must approve the request.

If you notice any discrepancies in the data, you can reject the request. The user will then have the opportunity to correct and resubmit it.
