Users with access to the Digital condominium


The register contains information about all registered users who have access to the current Digital Condominium.

Each user with access to the Digital Condominium can request access to information about a specific Digital Property.

All information access requests that triggered a verification security mechanism can be found in the section Pending Registration Requests.

If the request is approved, the user is granted access to the information for the specific Digital Property. If you reject the request, it is recorded in the registry of rejected requests and the user does not receive access to the requested information!

Information about all rejected registration requests can be found in the section Rejected Registration Requests.


As a building manager, you can send messages to each user and manage their access to the digital condominium by granting them special permissions. You can revoke a user’s access by using the remove user option.
